# サポートサイトのご案内

お客様向け管理システムに関する情報を、分かりやすくお届けするためのサポートサイトです。

## このサイトでご案内すること

- システムに関するお知らせ
- ご利用時に役立つ情報
- メンテナンスなどの重要なご連絡

本サイトでは、最新のお知らせを確認できるほか、必要に応じて利用ガイドを参照できます。詳細は、貴社のシステム管理者または担当窓口へお問い合わせください。

今後も皆様に必要な情報を、適切なタイミングでお届けしてまいります。
